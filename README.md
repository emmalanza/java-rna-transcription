# RNA Transcription

## Overview
This project implements a simple RNA transcription system in Java using **Test-Driven Development (TDD)**. The program converts a given DNA sequence into its RNA complement based on the following rules:

- `G` → `C`
- `C` → `G`
- `T` → `A`
- `A` → `U`

## Technologies Used
- **Java** 
- **JUnit 5** 
- **Maven**
